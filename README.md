# 🛠️ JSON Formatter + Validator

### A modern, client-side JSON toolkit that formats, perfectly validates, and aggressively auto-repairs broken JSON data structures in real-time.

<br/>

[![🚀 Live Demo](https://img.shields.io/badge/🚀%20Live%20Demo-json--formatter--30in30-6366f1?style=for-the-badge)](https://json-formatter-30in30.vercel.app)

## ⚙️ Features

<table>
  <tr>
    <td width="50%">
      <h3>✨ Real-time Validation</h3>
      Instantly parses your payload as you type, highlighting exact syntax error locations and lines so you can debug faulty JSON quicker.
    </td>
    <td width="50%">
      <h3>🛠️ Aggressive Auto-Repair</h3>
      Dynamically corrects advanced structural errors (like single quotes, unquoted keys, trailing commas, and JavaScript constants) completely automatically!
    </td>
  </tr>
  <tr>
    <td>
      <h3>🌳 Interactive Tree View</h3>
      Struggling with deeply nested APIs? Toggle on the Tree View to explore complex payloads through a nested, expandable hierarchy.
    </td>
    <td>
      <h3>🎨 Beautify & Minify</h3>
      Instantly prettify your JSON code with standard indentations, or squish it down into a single line for minimal storage sizes in 1 click.
    </td>
  </tr>
</table>

---

## 🔐 Privacy First

> **Your JSON payloads never leave your browser.**
> Everything runs strictly locally. Both Ace Editor rendering, JSON validations, and the regex auto-repair utilities compute locally on your CPU device, so your sensitive credentials or API exports are completely safe.
